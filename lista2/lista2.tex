\begin{enumerate}


\item O número de divisores positivos de $n$ é denotado por $d(n)$. Seja a fatoração em primos de $n = p_1^{\alpha_1} \times p_2^{\alpha_2} \times \ldots \times p_k^{\alpha_k}$ então o número de divisores é $d(n) = (\alpha_1+1) \times (\alpha_2+1) \times \ldots \times (\alpha_k+1)$.

Modifique o algoritmo de fatoração em primos para calcular $d(n)$.

\item O número de fatores primos diferentes de $n$ é denotado por $numDiff(n)$. Modifique o algoritmo de fatoração em primos para calcular $numDiff(n)$.

\item A soma de divisores positivos de $n$ é denotado por $\sigma(n)$. Desenvolva um algoritmo para calcular $\sigma(n)$.

\item O número de inteiros positivos menores que $n$ que são primos entre si com $n$ é denotado $\phi(n)$. Seja a fatoração em primos de $n = p_1^{\alpha_1} \times p_2^{\alpha_2} \times \ldots \times p_k^{\alpha_k}$ então o número de divisores é
\begin{equation}
\phi(n) = n(1 - \frac{1}{p_1})(1 - \frac{1}{p_2})\ldots (1 - \frac{1}{p_k})
\end{equation}

Modifique o algoritmo de fatoração para calcular $\phi(n)$.

\item O número de fatores primos diferentes de $n$ pode ser determinado para intervalo de inteiros $[0..MAXN]$ modificando o algoritmo de Crivo de Eratóstenes da seguinte maneira:

\begin{lstlisting}[frame=single,language=C]
vector<int> sieve_numdiff(int MAXN){
 vector<int> numDiff;
 numDiff.resize(MAXN+1,0);
 for(int i=2;i<=MAXN;i++){
  if(numDiff[i]==0){
   for(int j=i;j<=MAXN;j+=i)
    numDiff[j]++;
  }
 }
 return numDiff;
}
\end{lstlisting}

Modifique o algoritmo do Crivo de Eratóstenes para calcular $\phi(n)$ para todos os números no intervalo $[0 \ldots MAXN]$

\item O soma dos divisores de um número $n$ é denotado por $sumDiv(n)$. Modifique o algoritmo do Crivo de Eratóstenes para calcular $sumDiv(n)$ para todos os números no intervalor $[0 \ldots MAXN]$

\item O método de Horner consiste em reescrever um polinômio de forma a obter o valor de $p(x)$ tal que $p(x) = a_0 + a_1x+a_2*x^2+\ldots+a_nx^n$ em que $a_0,a_1,\ldots,a_n$ são os coeficientes do polinômio. Observe que que $p(x)$ pode ser reescrito da seguinte forma:
\begin{equation}
p(x) = a_0 + x(a_1+x(a_2*+\ldots+x(a_{n-1}+xa_n)))
\end{equation}

\begin{enumerate}
\item Implemente o método de Horner para a avaliação de um polinômio de grau $n$.
\item Utilize o método de Horner para descobrir o resto da divisão de número N de até 100 dígitos por um inteiro $M$.
\item Implemente um método que recebe um vetor binário representando um número na base 2 e devolve sua representação na base 10.
\end{enumerate}
